## 📖 نظرة عامة

هذا الموقع عبارة عن صفحة ويب تفاعلية (Single Page) تتيح للعميل اختيار طريقة الدفع المناسبة له من بين عدة وسائل (**Telda**، **Vodafone Cash**، **PayPal**)، وعرض التفاصيل والبيانات الخاصة بكل طريقة (رقم محفظة، بريد إلكتروني، تعليمات الدفع)، مع إمكانية نسخ البيانات مباشرة والتواصل لتأكيد عملية الدفع.

---

## ✨ المميزات

- 🎨 **تصميم عصري وناعم** بألوان بنفسجية متدرجة (Purple/Pink Gradient) وخطوط عربية أنيقة (Tajawal)
- 🃏 **بطاقات طرق دفع تفاعلية** يتم اختيارها بضغطة واحدة لعرض تفاصيلها فقط
- 🔄 **عرض ديناميكي للتفاصيل** — تظهر تفاصيل كل وسيلة دفع بانيميشن سلس عند اختيارها، وتختفي البقية
- 📋 **نسخ البيانات بضغطة زر** (رقم Vodafone Cash، بريد PayPal الإلكتروني) مع تأكيد بصري فوري
- 📞 **إجراءات مباشرة** للاتصال، أو الدفع عبر PayPal.Me مباشرة
- 🖼️ **معالجة ذكية للصور** — عند تعذر تحميل صورة وسيلة الدفع، يتم استبدالها تلقائيًا بصورة بديلة (Placeholder)
- 📢 **قسم تنويه بعد الدفع** يوضح للعميل كيفية إرسال إثبات الدفع عبر البريد الإلكتروني أو واتساب
- 📱 **تصميم متجاوب بالكامل (Responsive)** يعمل بسلاسة على الموبايل والتابلت والديسكتوب
- 🖱️ **دعم النقر المزدوج** لإلغاء اختيار وسيلة الدفع والعودة لرسالة الاختيار

---

## 💰 وسائل الدفع المتاحة

| الوسيلة | نوع البيانات المعروضة | الإجراءات المتاحة |
|---|---|---|
| **Telda** | تعليمات الدفع + صورة تأكيد | عرض التعليمات فقط |
| **Vodafone Cash** | رقم المحفظة | نسخ الرقم / الاتصال المباشر |
| **PayPal** | البريد الإلكتروني | نسخ البريد / الدفع عبر PayPal.Me |

---

## 🛠️ التقنيات المستخدمة

- **HTML5** لبناء هيكل الصفحة
- **CSS3** (Custom Properties, Flexbox, Animations, Media Queries) للتصميم والتجاوب
- **JavaScript (Vanilla)** لإدارة التفاعل، النسخ للحافظة، ومعالجة أخطاء تحميل الصور
- **Font Awesome 6.4.0** للأيقونات
- **Google Fonts** (Tajawal) لخط عربي واضح واحترافي

## 📂 هيكل المشروع

```
📁 Payment Methods
│
├── index.html              # الملف الرئيسي (HTML + CSS + JS)
├── Telda.jpg                # صورة تأكيد الدفع الخاصة بـ Telda
└── Sections-Images/          # مجلد صور وسائل الدفع
    ├── telda.jpg
    ├── vodafone.jpg
    └── paypal.jpg
```

> ⚠️ في حال عدم توفر إحدى الصور، سيقوم الموقع تلقائيًا بعرض صورة بديلة بدلاً منها.

## 🚀 طريقة التشغيل

1. حمّل الملفات أو استنسخ المشروع
2. تأكد من وجود مجلد **Sections-Images** وملف **Telda.jpg** بجانب `index.html`
3. افتح الملف مباشرة في المتصفح

```bash
# فتح الملف مباشرة
open index.html      # على macOS
start index.html     # على Windows
```

لا حاجة لأي سيرفر أو تثبيت إضافي — الموقع Static بالكامل.

---

## 🎯 كيفية الاستخدام

1. اختر وسيلة الدفع المناسبة من البطاقات المعروضة
2. سيتم عرض تفاصيل وسيلة الدفع (رقم، بريد إلكتروني، تعليمات) تلقائيًا مع تمرير سلس للأسفل
3. استخدم زر **النسخ** لنسخ البيانات، أو الأزرار المخصصة للاتصال أو الدفع المباشر
4. بعد إتمام الدفع، أرسل لقطة شاشة كإثبات عبر البريد الإلكتروني أو واتساب من قسم "تنويه هام بعد الدفع"
